Installation 
1. Git clone https://github.com/Produderox/DTP
Do all these below in vsc terminal 
2. pip install -r requirements.txt
3. python server.py

Features
Our website has 3 pages. A homepage to showcase the problem definition of food security, a statistic page
where user are able to explore our dataset and look at the various different graphs. Lastly a prediction page
where user input 4 variable rainfall,max temperature, min temperature and the crop type. It will output
a predicted crop-yield
